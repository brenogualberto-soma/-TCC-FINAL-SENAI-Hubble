
# TCC - Projeto Hubble

Esse foi o projeto elaborado por mim e meus colegas para o Trabalho de Conclusão de Curso em meu curso de Técnico em Informática para Internet (SENAI)

O projeto utiliza das linguagens de front-end (como HTML e CSS), o framework Bootstrap, e linguagens back-end (JavaScript).

# Aviso

Vale resaltar que o projeto foi elaborado para parecer que funciona - dado que a intenção original era funcionar apenas para a apresentação do trabalho em relação aos valiadores da banca. Ou seja, certas ações não funcionam. Ademais, o protejo possui alguns erros em relação às boas práticas de programação, bem como a falta de responsividade das páginas.
